## Print strings
* Print raw strings: print(r'c:\docs\navin') --> c:\docs\navin
  * without "r", "\n" will be a new line
* Escape charactor "\\": print('c:\docs\\navin') --> c:\docs\navin
* Repeart strings: print(2*'hello') --> hellohello
* _ + 3: '_' is previous output
